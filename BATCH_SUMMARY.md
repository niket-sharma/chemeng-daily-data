## How to Use

The generator script ([scripts/generate_entries.py](scripts/generate_entries.py)) allows you to:
- Add specific batches: `python scripts/generate_entries.py` (then enter batch numbers like `2,3,4`)
- Add all batches: `python scripts/generate_entries.py` (then enter `all`)

## Future Expansion

You can easily create more batches by:
1. Adding new `generate_batch_X()` functions to the script
2. Following the same pattern (20 compounds + 15 correlations + 15 constants)
3. Updating the main() function to include the new batch
